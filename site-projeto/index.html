    Perguntas Frequentes

    Quais as formas de pagamento?
    Quais plataformas de jogos vocês suportam?

    Atualmente, suportamos jogos para PlayStation e Xbox.

    Como faço para comprar um jogo no site?

    Para comprar um jogo, basta navegar em nossa loja, selecionar o jogo desejado e seguir as instruções de compra. Aceitamos cartões de crédito, PayPal e outras formas de pagamento.

    Vocês oferecem reembolso para jogos digitais?

    Sim, oferecemos reembolso para jogos digitais dentro de um prazo específico após a compra, sujeito aos nossos termos e condições de reembolso. Entre em contato com nosso suporte para obter mais informações.

    Quais são os requisitos mínimos de sistema para jogar um determinado jogo?

    Os requisitos mínimos de sistema para cada jogo estão listados na página do produto. Certifique-se de verificar esses requisitos antes de comprar ou baixar um jogo para garantir uma experiência de jogo adequada.

    Como faço para resolver problemas técnicos com um jogo?

    Se você encontrar problemas técnicos ao jogar um de nossos jogos, recomendamos consultar nossa seção de suporte ou entrar em contato com nossa equipe de suporte técnico. Estamos aqui para ajudar a resolver qualquer problema que você possa encontrar.
